The Open Microscopy Environment.

Analytical Microscopy- Anytime, Anywhere, Anyhow.

Some useful links:

  • The original whitepaper (pdf).  This whitepaper describes a CORBA distributed computing implementation.  The actual current implementation is to use a RDMS for shared memory, with connectors in xmlrpc (also ODBC, JDBC and Perl).  This whitepapaer is useful however as a big-picture description of our project.
  • A Presentation (pdf).  This presentation is not as out of date as the whitepaper, but is lacking in supporting text.  It does have diagrams of the RDMS schema (in UML), and screenshots!
  • xmlrpc documentation.  This is not very complete, and lacks examples for C, Java, etc.
  • XML DTD for OME dataset, and a jpeg diagram of same, and a pdf.
  • Another Presentaion (pdf).  This presentation was given at the Lab Robotics Ineterest Group (LRIG) May 23, 2001.  It has a simplified conceptual diagram of the data model, along with examples of how the API works. Screenshots of integration with Spotfire, and web-interface (OmniWeb).
  • OME-1.0-IR1.tar.gz  Download the source! This is the first internal distribution. First, look at the package dependencies to make sure you have everything you need. Then, look at the install document for additional installation help.

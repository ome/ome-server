-OME-

The Open Microscopy Environment.

Analytical Microscopy- Anytime, Anywhere, Anyhow.

Some useful literature describing OME:

  • The original whitepaper (pdf) written by Ilya Goldberg (igg@mit.edu) and Peter Sorger (psorger@MIT.EDU).
    This whitepaper describes a CORBA distributed computing implementation.  The actual current implementation is to use a RDBMS for shared memory, with connectors in xmlrpc (also ODBC, JDBC and Perl DBI).  This whitepapaer is useful however as a big-picture description of our project.
  • A Presentation (pdf) by Ilya Goldberg.
    This presentation is not as out of date as the whitepaper, but is lacking in supporting text.  It does have diagrams of the RDMS schema (in UML), and screenshots!
  • A subsequent whitepaper by Jason Swedlow (j.swedlow@dundee.ac.uk).
    Much of the missing narrative to go with the above presentation.
  • Another Presentation (pdf) by Ilya Goldberg.
    This presentation was given at the Lab Robotics Interest Group (LRIG) on May 23, 2001.  It has a simplified conceptual diagram of the data model, along with examples of how the API works. Screenshots of integration with Spotfire, and web-interface (OmniWeb).

Software:

  • OME-1.0-IR2.tar.gz  Download the source! This is the first internal distribution.
    First, look at the package dependencies to make sure you have everything you need.
    Then, look at the install document for additional installation help.
  • xmlrpc documentation.  This is not complete, and lacks examples for C, Java, etc.
  • XML DTD for the OME dataset, and a jpeg diagram of same, and a pdf.
